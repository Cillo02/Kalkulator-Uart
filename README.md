# UART-Kalkulator

Projekt für das Modul Programmierung mit C/C++.

## Projektbeschreibung

Der Benutzer gibt einen mathematischen Ausdruck über eine PC-Anwendung ein.
Der Ausdruck wird über UART an einen Mikrocontroller übertragen.
Der Mikrocontroller berechnet das Ergebnis und sendet es zurück.
